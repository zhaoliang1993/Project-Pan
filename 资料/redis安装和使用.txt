1.redis安装
$ wget http://download.redis.io/releases/redis-5.0.5.tar.gz
$ tar xzf redis-5.0.5.tar.gz
$ cd redis-5.0.5
$ make
$ make install

2.redis启动
  启动server: redis-server
  启动client: redis-cli
    测试是否连接
    *ping
        PONG
    *ping hello
        hello
  客户端关闭服务器
    shutdown（必须通过客户端连接到服务器）  

3.kedis中所有的数据都是键值对
  key: 字符串
  
4.redis常用数据类型
  字符串类型
      设置：set key value
      读取：get key
      批量设置：mset key1 value1 key2 value2 ...
      批量读取：mget key1 key2 ...
      显示所有键：keys *
  Hash类型
  List类型
  Set类型
  SortedSet类型zset
