1.redis安装
$ wget http://download.redis.io/releases/redis-5.0.5.tar.gz
$ tar xzf redis-5.0.5.tar.gz
$ cd redis-5.0.5
$ make
$ make install

2.redis启动
  启动server: redis-server
  启动client: redis-cli
    测试是否连接
    *ping
        PONG
    *ping hello
        hello
  客户端关闭服务器
    shutdown（必须通过客户端连接到服务器）  

3.kedis中所有的数据都是键值对
  key: 字符串
  
4.redis常用数据类型
  字符串类型
      设置：set key value
      读取：get key
      批量设置：mset key1 value1 key2 value2 ...
      批量读取：mget key1 key2 ...
      显示所有键：keys *
      数值自增：incr key
      数值自减：decr key
      字符串追加内容：append
      
  Hash类型
  List类型
    既可以安装队列操作，也可以安装栈操作
    左插入：lpush key value1 value2 value3 ...
    右插入：rpush key value1 value2 value3 ...
    遍历：lrange key 0 -1
    左侧弹出一个元素：lpop key
    右侧弹出一个元素：lpop key
    队列操作：
      队头出数据，队尾出数据
      rpush  lpop
    栈操作：
      只能操作栈顶
      lpush lpop
      或
      rpush rpop
    求list长度：llen key
    
  Set类型
  set集合没有排序，里面元素不能重复
  插入数据：sadd key value1 value2 value3 ...
  遍历数据：smembers key
  删除元素：srem key value1 value2 value3 ...
  差集：sdiff key1 key2   //key1-key2
  交集：sinter key1 key2
  并集：sunion key1 key2
  
  SortedSet类型zset
  插入：zadd key score1 value1 score2 value2 ...
  遍历：zrange key 0 -1（升序）
    zrange zset1 0 -1 withscores分数也打印出来
    zrevrange zset1 0 -1(降序)
